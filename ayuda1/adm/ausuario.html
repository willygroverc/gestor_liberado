AÑADIR USUARIO

Usted como administrador también puede añadir cuentas de usuario. En esta pantalla aparecen los campos que usted debe llenar para poder dar de alta a un usuario, los campos marcados con * son en los cuales usted debe tener cuidado de llenar tal y como son, porque si comente un error de sintaxis al dar de alta a un usuario, entonces este puede tener problemas futuros, cuando trate de ingresar a la mesa de ayuda .

 

Login; el login no puede contener mas de 6 caracteres, procure escribir el login en minúsculas, este campo tiene que estar lleno.

Password; el password no puede contener mas de 6 caracteres al igual que el login.

Tipo ; El tipo de usuario solo puede ser de 3 tipos, dependiendo de esta elección el usuario tendrá acceso a ciertas pantallas, los tipos pueden ser

    • Administrador

    • Técnico

    • Cliente

E-mail ; Introduzca en este campo el e-mail, del usuario a quien esté registrando, este e- mail debe ser válido, aunque puede ser vacio

Los campos NOMBRES, AP. PATERNO Y AP. MATERNO se refieren a los datos personales del usuario.

Cliente; se refiere al tipo de cliente al que se está dando de alta.

Entidad; se refiere a la entidad en la cual trabaja el cliente en caso de que el tipo de usuario sea cliente

Área ; se refiere al área en la cual trabaja el usuario.

Especialidad; se refiere a la especialidad que tiene el usuario dentro del área en la cual trabaja.

Cargo; se refiere al cargo que desempeña el usuario.

Teléfono ; se refiere al teléfono de referencia que tiene el usuario, este campo es opcional

Ext; en caso de que el usuario sea parte de otra entidad generalmente se hace referencia a una extensión, este campo es opcional

Ciudad ;se refiere a la ciudad en la que se encuentra el usuario.

Dirección; se refiere a la dirección domiciliaria del usuario,

si los campos han sido llenados adecuadamente presione el botón registrar y la mesa de ayuda, le mostrará la lista de usuarios actualizada.
