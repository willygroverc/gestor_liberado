CONTROL DE USUARIOS

Al hacer click sobre esta opción usted podrá ver la lista de control de usuarios:

Nro.; número correlativo que nos indica el número de usuario.

Usuario; nos indica el nombre del usuario.

Nro. De Identificadores; representa la cantidad de aplicaciones creadas para este usuario.

Insertar nueva aplicación; al hacer click sobre la opción NUEVA APLICACIÓN podrá crear una nueva, en la siguiente pantalla:

Donde podrá llenar los siguientes campos:

Nro.; número correlativo.

Nombre de Usuario; en este campo se puede llenar el nombre del usuario de la mesa de ayuda.

IDU; este campo nos indica el identificador del usuario.

Aplicación/Sistemas; en este campo debe especificar la aplicación con la cual trabajará el usuario.

Tipo de Acceso; en este campo debe especificar el tipo de acceso con el que contará el usuario, es decir especificar si por ejemplo tendrá acceso restringido.

Fecha Ingreso; nos indica la fecha en la que el usuario hizo su ingreso a la mesa de ayuda.

Dar de baja aplicaciones; al hacer click sobre esta opción, usted podrá anular una aplicación.

Imprimir; haciendo click en esta opción, usted podrá obtener un reporte de los datos del control de usuarios.

Haciendo clic sobre el botón IMPRIMIR, se genera un reporte de la lista de control de usuarios.
